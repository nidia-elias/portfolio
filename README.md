# Título del Proyecto

Portfolio
En la página web describo los lenguajes de programación que manejo, mi formación académica, hobbies, películas favoritas y habilidades. También podrán encontrar un formulario de contacto a través del cuál establecer una comunicación.

## Comenzando 🚀

 Desde arriba hacia abajo, lo primero que van a encontrar es el menú de navegación en el cuál se nombra las siguientes secciones: Sobre mí, Habilidades, Hobbies, Formación, Proyectos, Películas favoritas y Contacto. Si se hace click sobre cada una se mostrará centralmente la sección clickeada.

## Construido con 🛠️

Visual Studio Code
Lenguajes utilizados: HTML y CSS.
